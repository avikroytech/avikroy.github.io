# Avik Roy — Software Engineer & AI Enthusiast

Personal portfolio site built with Astro and Tailwind CSS showcasing projects, experience, education, and technical skills.

## Overview

This repository contains the source for my portfolio website. Content (name, bio, projects, skills, social links, and accent color) is configured in `src/config.ts`.

Key sections:
- About — personal bio
- Projects — featured projects with links and tech used
- Experience — internships and notable work
- Education — academic background and certifications
- Footer — contact links

## Tech Stack

- Astro — static site generator
- Tailwind CSS — utility-first styling
- TypeScript — configs and components

## Quickstart (Local Development)

1. Install dependencies

```bash
npm install
```

2. Run the dev server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

4. Preview the production build locally

```bash
npm run preview
```

These scripts are defined in `package.json`.
